== Description ==

DIV Layer Video Help puts video tutorials in the help section of the WordPress Admin Panel. This enables Administrators to see a video tutorial about the page they are on in their Admin Panel by clicking on the help tab.

== Installation ==

1. Extract div-layer-video-help.zip
2. Create a `div-layer-video-help` directory in `/wp-content/plugins/`
3. Upload all extracted files to the `div-layer-video-help` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Click on the help tab at the top of a page in the admin panel to watch a video about that page.
